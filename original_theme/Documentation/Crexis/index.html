There is a feature for the margin and paddings;

ml = margin-left
mr = margin-right
mt = margin-top
mb = margin-bottom

pl = padding-left
pr = padding-right
pt = padding-top
pb = padding-bottom

You can add margin or padding name and values, for example; 

<div class="mt-10">
	...
</div>

It'll have margin-top:10px; style automatically.


<div class="pl-82">
	...
</div>

It'll have padding-left:82px; style automatically.



<div class="pb-06">
	...
</div>

It'll have padding-bottom:6px; style automatically.


<div class="mr-09">
	...
</div>

It'll have margin-right:9px; style automatically.

You just need to write two numbers after typing abbreviations.

And so much more! You can see more shortcodes for classes top of the css/style.css file.

Super Slides, is a full screen slider. you can see it on full screen demos.

HTML;

Javascript:
Standart Using:
/* ==============================================
HOME PAGE IMAGE SLIDER (SUPER SLIDES)
=============================================== */

$('#fullscreen').superslides({
	play: 12000,
	animation_speed: 600,
	usekeyboard: false,
	animation: 'fade',
	inherit_height_from: window,
});

$('#fullscreen').superslides({
	play: false,
	animation_speed: 600,
	usekeyboard: false,
	animation: 'fade',
	inherit_height_from: window,
});

Box Sliders

We've included Flex Slider and Owl Carousel Slider
Owl Carousel Slider;
 $(".home-boxes").owlCarousel({
	items : 3,
	// Responsive Settings
	itemsDesktop : [1169,3],
	itemsDesktopSmall : [979,2],
	itemsTablet : [600,1],
	itemsTabletSmall : false,
	itemsMobile : [560,1],
	// End Responsive Settings
	slideSpeed : 400
});

You can use Carousel Slider with just classes. For example; 

<div class="boxes boxes-type-4 box-carousel three-items clearfix">
	<div class="box">
		...
	</div>
</div>

You just need add "box-carousel" class and item number. You can add, "three-items", "four-items", "five-items", "six-items" for item numbers. And you can add box-carousel-dragable class for dragable boxes.

Note; You can edit these options in the js/plugins.js file, for example; 

/* ==============================================
CAROUSEL SLIDER FOR BOXES / 5 ITEMS
=============================================== */  

    var owlBox = $(".box-carousel.five-items");
    
    // Add .box-carousel class to boxes div
    owlBox.owlCarousel({
        items : 5,
        // Responsive Settings
        itemsDesktop : [1169,5],
        itemsDesktopSmall : [979,4],
        itemsTablet : [640,3],
        itemsTabletSmall : false,
        itemsMobile : [560,3],
        // End Responsive Settings
        mouseDrag : true,
        pagination : false,
        navigation : false,
        touchDrag : true,
        slideSpeed : 400
    });
Flex Slider;
/* ==============================================
CUSTOM IMAGE SLIDER

$('.basic_slider').flexslider({

Contact Forms

You can use your domain's settings or Gmail settings.

Edition;
$mail->Host     = "smtp.yourdomain.com"; 			 // SMTP server
$mail->SMTPAuth = true;    					 // turn on SMTP authentication
$mail->Username = "no-reply@yourdomain.com"; 		         // SMTP username
$mail->Password = "secret";					 // Password

$mail->From     = "no-reply@yourdomain.com";			 // SMTP username again
$mail->AddAddress("your-email@domain.com");		  	 // Your Adress
$mail->Subject  =  "New mail from Crexis !";
Gmail;
$mail->IsSMTP();                                 				 // send via SMTP
$mail->Host     = "smtp.gmail.com"; 						 // SMTP server for Gmail
$mail->SMTPAuth = true;    	
$mail->SMTPSecure = 'tls';								
$mail->Username = "mailer-address@gmail.com"; 				 	 // Your Mailer Gmail Address
$mail->Password = "mailer-address-password";					 // Password
$mail->Port = 587;	
$mail->From     = "mailer-address@gmail.com";				 	 // Your Mailer Gmail Address (Same With Username)
$mail->AddAddress("your-address@domain.com");			  		 // Your Address
$mail->Subject  =  "New mail from Crexis !";
